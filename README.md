# Login Form

A clean, responsive login form built using plain HTML, CSS, and JavaScript.

![Login Form Screenshot](./Screenshot%202024-08-01%20140809.png)

Description
-----------
This repository contains a simple and stylish login form UI that demonstrates layout, form validation, and basic interaction using vanilla HTML, CSS, and JavaScript. It ships with all assets required to run locally.

Features
--------
- Responsive layout that looks good on desktop and mobile
- Client-side validation for required fields
- Subtle UI effects and accessible labels
- Easy to customize styles and assets

Built With
----------
- HTML
- CSS
- JavaScript

How to run
----------
1. Clone or download this repository.
2. Open `index.html` in your web browser (no build step required).

Files of interest
-----------------
- `index.html` — main page containing the form markup
- `style.css` — styling for the form and page
- `3d-cartoon-night-sky.jpg` — background image used by the page
- `Screenshot 2024-08-01 140809.png` — screenshot shown above

Customization
-------------
- To change colors or spacing, edit `style.css`.
- To replace the background image, swap `3d-cartoon-night-sky.jpg` and update the CSS if needed.
